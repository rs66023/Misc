HTML Lists

Lists as expected are ways to display information in a neat and orderly fashion, HTML has multiple ways with which lists can be displayed.

Lists are displayed with nested tags. That means a tag within another tag. Ordered lists go with the 'ol' tag, and unordered go with 'ul', with individual list items in both displayed with 'li'

Different types of lists in HTML

Unordered lists

Unordered lists, created with a ul tag show a list without a specific order, ie a bulleted list.
The 'list-style-type' element is used to define the type of bullet list.

A nested list is one with list items contained within other list items.
This can be achieved by opening another ul and li tag withing an existing li.
