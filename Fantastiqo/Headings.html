HTML Headings

Headings are used to provide titles or subtitles that we want to use for a page.

There are a range of 6 headings, from h1, to h6. The size of the text reduces respectively from largest to smallest.

What are Headers used for?

Search engines use the headings to index the structure and content of your web pages. Users often skim a page by its headings. It is important to use headings to show the document structure.
'h1' headings should be used for main headings,
followed by 'h2' headings,
then the less important 'h3', and so on.

Important

Use the h tags only for headings, don't use them to make text BIG or BOLD
