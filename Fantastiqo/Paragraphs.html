HTML Paragraphs

A paragraph is a unit of writing in a larger body of work. A paragraph expresses a particular topic or theme.

In HTML, paragraphs are denoted with a 'p' tag. Each new element starts on a new line and displays it's text as a paragraph, and browsers automatically add the appropriate spacing before and after it.

One cannot be exactly sure how HTML will be displayed. The text gets resized according to the screen size, so different sizes give different results. Text cannot be manipulated by adding extra spaces and lines in the text, the 'p'tag automatically removes any spacing and lines before displaying it.

At the same time,the hr tag, can be used to display horizontal lines as dividers.

This is a sample paragraph divided with hr tags

The hr tags, along with br tags can be used to split up individual paragraphs.

Similarly, br tags
can be used to split up the paragraphs with lines.

Each tag adds another line and they can be stacked ontop of one another.
