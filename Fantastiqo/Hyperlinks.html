HTML Links

HTML Links are how we navigate from one page to another.

HTML links are Hyperlinks. They allow us to navigate from one page to another by clicking the link.

The 'a' tag is used to create links, it's paired with the attribute href within it, that defines where the link takes us.For example: href="Google.com"

A secondary attribute is the 'target' attribute, that specifies where to open the linked page. There are 4 different ways it can go

  1. _self - Opens the document in the same tab as it was clicked on. This is the default until specified otherwise.
  2. _blank - Opens the document in a new window or tab
  3. _parent - Opens the document in the parent frame
  4. -top - Opens the document in the full body of the window

Any text in the a tag will be the text of the link it's made for.
