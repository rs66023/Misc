CSS

CSS stands for Cascading Style Sheets, it's used to format the layout of a webpage.

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

There are three ways to add CSS into HTML files

  1. Inline - by using the style attribute inside HTML elements
  2. Internal - by using a 'style' element in the 'head' section
  3. External - by using a 'link' element to link to an external CSS file

Colours, Fonts and Sizes

Some of the commonly used CSS Properties include:

  1. The CSS color property defines the text color to be used.
  2. The CSS font-family property defines the font to be used.
  3. The CSS font-size property defines the text size to be used.

CSS Borders

The CSS border property defines a border around an HTML element.

CSS Padding

The CSS padding property defines a padding (space) between the text and the border.

CSS Margin

The CSS margin property defines a margin (space) outside the border.
